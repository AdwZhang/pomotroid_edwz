# 任务备注功能

## 概述
为专注轮次添加可选的任务备注能力。开启后，用户可以描述当前正在做什么，描述会随完成的 session 记录到数据库中，方便回看。

## 动机
用户想知道时间具体花在了什么上面。目前 session 只记录时间数据，没有工作内容的上下文信息。

## 范围
- 设置开关：`task_notes_enabled`（默认关闭）
- 主计时器 UI 文本输入框（round label 下方，controls 上方）— 仅在 work 轮次且开启时显示
- 视觉状态区分：未聚焦时透明无边框（像标签），悬停时微现背景，聚焦时显示边框+背景
- "当前任务备注"跨轮次保留，直到用户手动修改/清空
- session 完成时取当前最新值记录到 DB
- 日视图统计页面展示带备注的会话列表
- 100 字符限制
- Compact 模式（<300px 窗口）：隐藏文本框（空间太小，会挤压时钟显示）
- 休息轮次：文本框不显示，不记录备注

## 范围外
- 任务列表管理
- 分类/标签系统
- 周/年视图的备注展示
